## <kbd>function</kbd> `compute_mo_gae`

```python
compute_mo_gae(
    truncation: jax.Array,
    termination: jax.Array,
    rewards: jax.Array,
    values: jax.Array,
    bootstrap_value: jax.Array,
    lambda_: float = 1.0,
    discount: float = 0.99
)
```

Calculates the Generalized Advantage Estimation (GAE) in the multi-objective setting. T represents time, B is the batch size, and  M is the number of objectives 



**Args:**
 
 - <b>`truncation [T, B]`</b>:  truncation signal. 
 - <b>`termination [T, B]`</b>:  termination signal. 
 - <b>`rewards [T, B, M]`</b>:  multi-objective rewards generated by following the policy. 
 - <b>`values [T, B, M]`</b>:  multi-objective value function estimates wrt. the target policy. 
 - <b>`bootstrap_value [B, M]`</b>:  multi-objective value function estimate at time T. 
 - <b>`lambda_`</b>:  Mix between 1-step (lambda_=0) and n-step (lambda_=1). Defaults to lambda_=1. 
 - <b>`discount`</b>:  TD discount. 



**Returns:**
 
 - <b>`Values [T, B, M]`</b>:  can be used as target to train a baseline (V(x_t) - vs_t)^2. 
 - <b>`Advantages [T, B, M]`</b>:  for policy loss

## <kbd>function</kbd> `compute_mo_ppo_loss`

```python
compute_mo_ppo_loss(
    params: moplayground.moppo.losses.MOPPONetworkParams,
    normalizer_params: Any,
    data: moplayground.moppo.acting.MultiObjectiveTransition,
    rng: jax.Array,
    moppo_network: moplayground.moppo.factory.MOPPONetworks,
    entropy_cost: float = 0.0001,
    discounting: float = 0.9,
    reward_scaling: float = 1.0,
    gae_lambda: float = 0.95,
    clipping_epsilon: float = 0.3,
    normalize_advantage: bool = True
) → Tuple[jax.Array, Mapping[str, jax.Array]]
```

Computes PPO loss. 



**Args:**
 
 - <b>`params`</b>:  Network parameters, 
 - <b>`normalizer_params`</b>:  Parameters of the normalizer. 
 - <b>`data`</b>:  Transition that with leading dimension [B, T]. extra fields required  are ['state_extras']['truncation'] ['policy_extras']['raw_action']  ['policy_extras']['log_prob'] 
 - <b>`rng`</b>:  Random key 
 - <b>`ppo_network`</b>:  PPO networks. 
 - <b>`entropy_cost`</b>:  entropy cost. 
 - <b>`discounting`</b>:  discounting, 
 - <b>`reward_scaling`</b>:  reward multiplier. 
 - <b>`gae_lambda`</b>:  General advantage estimation lambda. 
 - <b>`clipping_epsilon`</b>:  Policy loss clipping epsilon 
 - <b>`normalize_advantage`</b>:  whether to normalize advantage estimate 



**Returns:**
 A tuple (loss, metrics)
